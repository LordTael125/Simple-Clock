# Simple Clock

A stylish, transparent, and round analog clock widget for your desktop.

## Features
- **Transparent & Frameless**: A clean, circular design that blends into your desktop.
- **Always on Top**: Keeps the time visible above other windows.
- **Interactive**:
    - **Move**: Drag anywhere on the clock face.
    - **Resize**: Drag the grip at the bottom-right.
    - **Close**: Click the red dot at the top-right or use the right-click menu.
- **Dual Display**: Analog hands and a 7-segment digital display (HH:MM).
- **Dynamic Transparency**: Becomes more opaque when focused.

## Installation & Usage

### Requirements
- Python 3.x
- PyQt5

### Running from Source
```bash
pip install PyQt5
python main.py
```
### Note
- To keep window on top of the app make sure to apply the properties through the window compositor
